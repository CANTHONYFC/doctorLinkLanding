Preguntas Frecuentes

¿Qué es Dr. Link?

Dr. Link es una plataforma integral de gestión médica que permite a profesionales de la salud y clínicas organizar su operación administrativa y clínica desde un solo lugar. Facilita la atención al paciente, automatiza procesos y mejora la eficiencia del consultorio.

¿Cómo funciona Dr. Link?

Funciona como un sistema en la nube, accesible desde cualquier dispositivo con conexión a internet. Al registrarte, puedes comenzar a gestionar tus pacientes, agendar citas, enviar recordatorios, cargar historias clínicas, emitir comprobantes y acceder a reportes.

¿Cuánto cuesta Dr. Link?

Dr. Link ofrece tres planes: Básico, Gold y Premium, desde $29 USD al mes. Cada plan incluye diferentes beneficios según el número de usuarios, pacientes, funcionalidades y nivel de automatización. Puedes ver el detalle completo en nuestra sección de precios.

¿Dr. Link se puede usar gratis?

No contamos con una versión gratuita permanente, pero ofrecemos una demostración sin costo para que puedas conocer el sistema antes de contratarlo. Además, algunas promociones pueden incluir períodos de prueba.

¿Dr. Link es un software para dentistas?

Sí. Dr. Link es ideal para consultorios odontológicos, ya que permite gestionar historias clínicas, tratamientos por odontograma, citas por especialidad, fotografías de seguimiento, recordatorios y mucho más.

¿Dr. Link es un software para medicina estética?

Sí. Nuestra plataforma está adaptada a las necesidades de la medicina estética: manejo de fichas personalizadas, fotografías antes/después, control de insumos, y agenda por procedimientos.

Si trabajo solo, ¿Dr. Link me sirve?

Claro. El plan Básico está pensado para profesionales independientes que gestionan su propia consulta. Podrás ahorrar tiempo y tener todo ordenado desde un solo sistema.

¿Dr. Link funciona para clínicas grandes con varias sedes?

Sí. El sistema permite configurar varias sedes, gestionar múltiples usuarios y especialidades, llevar reportes separados por local y centralizar toda la operación desde una sola cuenta.

¿Dr. Link es un CRM?

Dr. Link incluye funciones propias de un CRM: segmentación de pacientes, envío automatizado de mensajes, historial de atenciones y reportes de comportamiento, lo que te permite mejorar tu relación con los pacientes.

¿Dr. Link es un ERP?

Aunque no es un ERP completo, incorpora varios módulos de gestión: facturación, inventario, reportes financieros, control de usuarios y más. Es una solución muy completa para centros médicos.

¿Dr. Link ofrece descuentos especiales?

Sí. Contamos con promociones por pago anual, descuentos por cantidad de usuarios, y alianzas especiales. Puedes consultar las ofertas vigentes escribiendo a nuestro equipo comercial.

¿Dr. Link funciona en mi país?

Dr. Link es una plataforma en la nube disponible para cualquier país. Actualmente operamos principalmente en Latinoamérica, y adaptamos módulos según requerimientos locales.

¿Dr. Link cuenta con Facturación Electrónica?

Sí. En países donde es obligatorio, ofrecemos integraciones para emitir comprobantes electrónicos de manera automática, como boletas, facturas y notas de crédito.

¿Mi asistente y personal administrativo pueden usar Dr. Link?

Por supuesto. Puedes crear diferentes usuarios con permisos personalizados para recepcionistas, asistentes, facturadores, entre otros. Cada uno tendrá acceso solo a lo necesario.

¿Dr. Link cuenta con soporte gratuito?

Sí. Todos los planes incluyen soporte vía correo y chat. Además, ofrecemos sesiones de capacitación virtual y recursos de ayuda en línea. Nuestro objetivo es que aproveches el sistema al máximo.
